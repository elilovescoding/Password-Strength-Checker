# Password Strength Checker

This is a Python program that evaluates the strength of a password based on multiple security criteria.

## Features
- Checks if password length is at least 8 characters
- Detects uppercase letters
- Detects numbers
- Detects special characters
- Provides a strength rating (Weak, Medium, Strong)

## Technologies Used
- Python

## How to Run
1. Run the Python file
2. Enter a password when prompted
3. The program will return a strength rating

## Example Output
Enter a password: Abc123!
Medium password  
Password strength score: 3 /4
